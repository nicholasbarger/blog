In addition, a web-based dispatching application was created to use internally to assist in scheduling and executing service calls.

This project was done long before the now dominant jQuery draggable functionality but allowed for full drag and drop of scheduled calls to the appropriate service technician. Each drop event registered a callback to the database to save the associated data and if necessary send a notification of the job assignment.
